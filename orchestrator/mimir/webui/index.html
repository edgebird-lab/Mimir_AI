🧑‍💻 Coding — Mimir programmiert für dich, isoliert in einer eigenen VM

Hier arbeitet Mimir selbstständig an echtem Code — mit Shell, Git, Build- und Testwerkzeugen, komplett abgeschottet vom Rest deines Rechners:

  1. ① Projekt wählen — ein bestehendes aus der Liste, oder + Neues Projekt für ein leeres.
  2. ② Beschreiben, was gebaut werden soll — unten im großen Feld, z. B. „Baue einen Taschenrechner mit Java Swing (Grundrechenarten)."
  3. ③ 🤖 Los — Mimir öffnet die VM (falls nötig) und arbeitet autonom: bearbeitet Dateien, führt Tests aus. Du siehst rechts unten live, was passiert.
  4. ⇪ Merge-back übernimmt das Ergebnis erst nach deiner Prüfung/Freigabe in den „📚 Bibliothek"-Export-Ordner. Links siehst du jederzeit die Dateien im Workspace.
