# Aufgabe: YouTube Downloader
Schreibe ein Python-CLI (yt_dl.py), das eine oder mehrere YouTube-URLs entgegennimmt und das Video
herunterlädt — oder mit --mp3 nur die Audiospur als MP3 extrahiert (via yt-dlp + ffmpeg).
Mit Tests (test_yt_dl.py), die OHNE Netzwerk laufen (Optionen-/Format-Logik prüfen).
